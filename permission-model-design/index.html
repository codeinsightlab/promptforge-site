# 分析约束 / 执行约束

1. 必须区分功能权限、接口权限和数据权限。
2. 必须覆盖水平越权、角色膨胀、超级管理员和审计。
3. 不要设计过度复杂的权限模型，优先可落地。
4. 信息不足时最多追问 3 个关键问题。
